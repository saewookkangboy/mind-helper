# Cursor 에이전트 팀

Fortune Mate는 **에이전트 팀** 중심 **오케스트레이션**으로 서비스 최적화를 운영합니다.

## 팀 구성

| 에이전트 | 파일 | 사용 시점 |
|----------|------|-----------|
| **Orchestrator** | [orchestrator.md](./orchestrator.md) | 서비스 최적화, 전체 개선, agent team으로 진행 |
| **Dev Agent Kit** | [dev-agent-kit.md](./dev-agent-kit.md) | 역할 기반 개발, SEO/GEO/AIO, 사양·To-do |
| **Spec & To-do** | [spec-and-todo.md](./spec-and-todo.md) | 사양 문서, To-do·마일스톤만 다룰 때 |

## 빠른 선택 가이드

- **"서비스 최적화해줘"**, **"agent team으로 진행"** → **Orchestrator**
- **"SEO 점검"**, **"역할별로 개선"** → **Dev Agent Kit**
